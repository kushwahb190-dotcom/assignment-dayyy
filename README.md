# Smart Campus Management System (Java)

## 📌 Problem Statement
This project is a Smart Campus System developed in Java to manage students, courses, and enrollments.

## 🚀 Features
- Add Students
- Add Courses
- Enroll Students
- View Students
- View Enrollments
- Multithreading for processing enrollment
- Exception Handling (Custom Exception)
- File Handling (Save Data)

## 🛠️ Technologies Used
- Java
- OOP Concepts
- Collections (HashMap, ArrayList)
- Multithreading
- Exception Handling
- File Handling

## 📊 Output
Menu Driven Console Application

## 🧠 Concepts Used
- Classes & Objects
- Constructor
- HashMap for storage
- Thread class
- Custom Exception
